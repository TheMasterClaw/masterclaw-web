# MasterClaw Command Center

Web-based hackathon deadline tracker and project dashboard.

## Features

- ⏰ **Live Countdown Timers** - Real-time deadline tracking
- 📊 **Project Status Dashboard** - Visual readiness indicators  
- 🚀 **Quick Actions** - One-click access to submissions
- 🔔 **Deadline Alerts** - Automatic warnings (6h, 1h, overdue)
- 📜 **Activity Log** - System event tracking

## Live Demo

**Deployed at:** https://masterclaw-web.vercel.app

## Local Development

```bash
# Clone
git clone https://github.com/TheMasterClaw/masterclaw-web.git
cd masterclaw-web

# Serve locally
npx serve .
# or
python3 -m http.server 8080
```
